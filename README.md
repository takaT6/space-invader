Click to install http://www.cc.kyoto-su.ac.jp/~ogihara/Hg/products/HandyGraphic0.83.dmg.zip

# Usage

after activating HandyGraphic...
```bash
*build
hgcc -o SpaceInvader SpaceInvader.c

*run
./SpaceInvader
```
